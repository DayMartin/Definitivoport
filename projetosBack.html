Machine Learning - Árvore de Decisão - Python - Scikit-learn - NumPy - Pandas - Matplotlib

Compreendendo o processo por trás da árvore de decisão e criando um modelo utilizando um dataset disponibilizado pelo Hospital Einstein no SARS-COV-2, com informações de pacientes que tiveram entrada no hospital, fizeram o teste e se submeteram a outros exames adicionais, tais como 'Hemoglobina', 'Leucócitos', 'Basófilos', 'Proteína C reativa mg/dL'. São esses exames que utilizei para criar o modelo atual. A partir dos exames realizados, o modelo prevê se o paciente pode ter ou não o Covid-19. OBS: Este modelo foi utilizado apenas para estudos. O dataset contém quase 6 mil pacientes, porém, apenas 302 realizaram os exames adicionais, um número muito pequeno para a utilização desse modelo em produção. No entanto, para fins didáticos, o dataset é bastante adequado.
